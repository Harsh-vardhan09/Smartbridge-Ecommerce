# ShopEz

- This is an shopping point for all the items a person wants
- This is an MERN Program which uses all database to store and work data.
- This has cart, user, admin and auth using jwt

## ⚙️ Installation & Setup

1. ***Clone the repo:-
```bash
git clone <Repolink>
//open ShopEz
```

2. ***Setup Client:-
```bash
cd client
npm install 
```

3. ***Setup server:-
```bash
cd server
npm i
```

4. Root server:-
```bash
//enter root by doing cd..
npm i
npm run dev
```
